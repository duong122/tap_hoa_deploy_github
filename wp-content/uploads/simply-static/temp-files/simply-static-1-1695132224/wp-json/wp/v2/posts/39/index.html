Ngo\u00e0i ra, c\u1ea5p ph\u00f3 c\u1ee7a c\u00e1c t\u1ed5 ch\u1ee9c n\u00e0y c\u0169ng \u0111\u01b0\u1ee3c h\u01b0\u1edfng m\u1ee9c th\u00f9 lao 50-100 tri\u1ec7u \u0111\u1ed3ng\/th\u00e1ng.<\/p>\n
